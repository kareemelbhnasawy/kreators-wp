# Kreators Theme - Image Assets

This directory contains image assets for the Kreators theme.

## Required Images

1. **screenshot.png** (1200x900 pixels recommended)
   - Place in the theme root directory (`/themes/kreators/screenshot.png`)
   - This is displayed in WordPress admin when selecting themes

2. **creator-1.jpg, creator-2.jpg, creator-3.jpg**
   - Place in `/assets/images/`
   - Used for the homepage hero section visual
   - Recommended size: 400x560 pixels (portrait orientation)
   - Should feature diverse, professional creator photos

3. **logo.png / logo.svg**
   - Place in `/assets/images/`
   - Theme logo for header (if not using custom logo from Customizer)
   - Recommended size: 300x80 pixels max

4. **placeholder.jpg**
   - Default placeholder for posts without featured images
   - Recommended size: 600x400 pixels

## Creating a Screenshot

To create a proper theme screenshot:
1. Take a screenshot of your homepage
2. Crop to 1200x900 pixels
3. Save as PNG
4. Place in the theme root as `screenshot.png`
